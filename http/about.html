About Pastebin

This is a pastebin proxy that interacts with the backend at https://pastebin.aosc.io to do what a pastebin is supposed to do. Compare to the backend, this offers better compatibility, accessibility, and speed, in addition to offering a UI that more closely resembles the AOSC main site.
